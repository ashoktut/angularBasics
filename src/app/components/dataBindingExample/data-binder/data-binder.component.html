What is Data Binding ?

    Data binding is the communication between the Typescript component and the HTML template

4 ways of Data Binding

  • Method 1 & 2 are from the ts component to the HTML template
  • Method 3 is from the HTML template to the ts component
  • Method 4 is 2 way Binding between both Html and ts

Method: 1 String Interpolation

  • Used to usually output some value to the HTML template
    My name is: {{name}}

Method: 2 Property Binding

  • Property Binding uses square brackets [ ] to change any element's property declared in ts to HTML
  • In this case the src property of the img element to display the angular logo:

Method: 3 Event Binding

  • Event Binding uses paranthesis {{specialChar}}
  • Can be used to listen for user actions like keystrokes, mouse movements, click etc
  • In this case the the keystrokes of the text in textbox is sent to ts and displayed in console.log

Method: 4 Two-Way Binding

  • Two-Way Binding uses the banana in the box method => ( {{specialChar}} )
  • Binding happens in both the directions simultaneously between HTML and TS
  • In this case whatever typed in txtBox shows in span and whatever changed in declaration TS shows in textbox
{{myFavFood}}
